## Read Before Starting A Project

### Coding

**GitHub Documentation** - Essential for anyone not familiar with Git (central to any coding project execution)
- [GitHub Flow](https://docs.github.com/en/get-started/using-github/github-flow)
- [GitHub Projects](https://docs.github.com/en/issues/planning-and-tracking-with-projects) - Planning, tracking and delivering projects with tasks and timelines

### Research

**[How to Read a Paper](http://ccr.sigcomm.org/online/files/p83-keshavA.pdf)** - 2-page guide on effective research paper reading

### Machine Learning

**[Rules of Machine Learning | Google for Developers](https://developers.google.com/machine-learning/guides/rules-of-ml)** - How to build good ML-based algorithms. TLDR: Simpler is Better.

### Infrastructure

**[Vast.ai](https://vast.ai/)** - Affordable and easy to set up GPU server renting

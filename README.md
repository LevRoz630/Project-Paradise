# Project Paradise

Website for [project-paradise.co.uk](https://project-paradise.co.uk)

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

View at http://localhost:4000

## Structure

```
├── index.md              # Homepage
├── projects/             # Project pages
├── competitions/         # Competitions and write-ups
├── team/                 # Member and supervisor profiles
├── resources/            # Learning materials
├── _layouts/             # Page templates
├── _includes/            # Reusable components
├── _data/                # YAML data files
├── assets/css/           # Stylesheets
└── assets/images/        # Images
```

## Adding Content

See [Adding Pages](https://project-paradise.co.uk/resources/adding-pages/) on the site.

## Deployment

Push to `main` and GitHub Pages builds automatically.
